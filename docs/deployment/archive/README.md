# アーカイブ

## 📋 概要

古いトラブルシューティング記録や完了済みの問題の詳細記録を保存しています。

## 📁 ディレクトリ構成

- **auth/** - 認証関連の古い記録（約60個）
- **nginx/** - Nginx関連の古い記録（約40個）
- **deployment/** - デプロイメント関連の古い記録（約30個）
- **security/** - セキュリティ関連の古い記録（約20個）
- **external-access/** - 外部アクセス関連の古い記録（約10個）

## 🔍 参照方法

これらのドキュメントは過去の問題解決の記録として保存されています。同様の問題が発生した場合の参考として使用できます。

### 認証関連 (`auth/`)

認証機能の実装・統合に関する古い記録：
- `DASHBOARD_AUTH_*` - ダッシュボード認証関連
- `MEETING_MINUTES_AUTH_*` - Meeting Minutes認証関連
- `YOUTUBE_TO_NOTION_AUTH_*` - YouTube to Notion認証関連
- `DOCUMENT_AUTOMATION_MONITORING_AUTH_*` - Document Automation認証関連

### Nginx関連 (`nginx/`)

Nginx Proxy Managerの設定・トラブルシューティングに関する古い記録：
- `NGINX_PROXY_MANAGER_*` - Nginx Proxy Manager関連
- `NGINX_*` - Nginx全般

### デプロイメント関連 (`deployment/`)

各プロジェクトのデプロイメントに関する古い記録：
- `AMAZON_ANALYTICS_*` - Amazon Analytics関連
- `DOCUMENT_AUTOMATION_*` - Document Automation関連
- `MEETING_MINUTES_*` - Meeting Minutes関連
- `YOUTUBE_TO_NOTION_*` - YouTube to Notion関連
- `STATIC_FILES_404_*` - 静的ファイル404エラー関連
- `SUBFOLDER_*` - サブフォルダ対応関連

### セキュリティ関連 (`security/`)

セキュリティ設定・トラブルシューティングに関する古い記録：
- `FIREWALL_*` - ファイアウォール関連
- `FAIL2BAN_*` - Fail2Ban関連
- `SECURITY_*` - セキュリティ全般

### 外部アクセス関連 (`external-access/`)

外部アクセス設定に関する古い記録：
- `EXTERNAL_ACCESS_*` - 外部アクセス全般
