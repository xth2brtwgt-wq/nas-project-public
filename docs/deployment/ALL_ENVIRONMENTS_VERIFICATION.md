# 全環境の状態確認

## 📋 概要

ローカル環境とNAS環境の両方で、全環境が揃っている状態か確認します。

## 🔍 確認項目

### 1. Gitの状態確認

- 最新コミットの確認
- 未コミットファイルの確認
- リモートとの同期状況

### 2. プロジェクトフォルダ内の生成物確認

- ログファイルの確認
- データファイルの確認
- 生成物ディレクトリの確認

### 3. 一時的なバックアップファイル確認

- 一時的なバックアップファイルの確認

### 4. コード修正の確認（NAS環境のみ）

- 各プロジェクトのログ設定が正しく修正されているか

### 5. コンテナの状態確認（NAS環境のみ）

- 各プロジェクトのコンテナが正常に起動しているか

### 6. nas-project-data配下のログ確認（NAS環境のみ）

- 各プロジェクトのログが正しい場所に保存されているか

### 7. 環境変数ファイルの確認

- 各プロジェクトの`.env`ファイルが存在するか

## 🚀 確認方法

### ローカル環境

```bash
# 確認スクリプトを実行
./scripts/verify-all-environments.sh
```

### NAS環境

```bash
# NAS環境にSSH接続
ssh -p 23456 AdminUser@192.168.68.110

# 確認スクリプトを実行
cd ~/nas-project
./scripts/verify-all-environments.sh
```

## ✅ 期待される結果

### ローカル環境

- ✅ 最新コミットが表示される
- ✅ 未コミットファイルなし
- ✅ リモートと同期済み
- ✅ プロジェクトフォルダ内に生成物なし
- ✅ 一時的なバックアップファイルなし
- ✅ 環境変数ファイルが存在

### NAS環境

- ✅ 最新コミットが表示される
- ✅ 未コミットファイルなし
- ✅ リモートと同期済み
- ✅ プロジェクトフォルダ内に生成物なし
- ✅ 一時的なバックアップファイルなし
- ✅ コード修正が反映されている
- ✅ すべてのコンテナが正常に起動している
- ✅ nas-project-data配下にログファイルが保存されている
- ✅ 環境変数ファイルが存在
