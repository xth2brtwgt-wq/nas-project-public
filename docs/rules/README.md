## 使い方

### AIアシスタント（Cursor）
AIアシスタントは、`.cursorrules`ファイルを参照し、必要に応じて各Markdownファイルを検索して読み取ります。

### 開発者
特定のルールを確認したい場合は、上記のファイルを直接参照してください。また、`.cursorrules`ファイルからも概要を確認できます。

## ルールの更新

各ルールファイルは独立して更新できます。ルールを変更する際は：
1. 該当するMarkdownファイルを更新
2. 必要に応じて`.cursorrules`の概要も更新
3. 変更内容をコミット

### 過去のチャットを再開した場合のルール適用

**一般的な動作**：

1. **ルールファイルは最新の状態で読み込まれる**
   - 過去のチャットを再開した場合でも、`.cursorrules`ファイルと`docs/rules/`のファイルは、**チャットセッション開始時に最新の状態で読み込まれます**
   - つまり、チャット後にルールを変更した場合、そのチャットを再開すると**最新のルールが適用されます**

2. **チャットのコンテキストは保持される**
   - 過去のチャットの会話履歴は保持されますが、ルールファイルは最新の状態で読み込まれます
   - これにより、過去のチャットの文脈を保ちながら、最新のルールに基づいて回答されます

3. **推奨事項**
   - ルールを変更した後、過去のチャットを再開する場合は、必要に応じて「ルールが更新されました」と明示的に伝えることを推奨します
   - 重要なルール変更がある場合は、新しいチャットを開始することを推奨します

**注意事項**：
- ルールファイルの読み込みタイミングは、Cursorの実装に依存します
- 確実に最新のルールを適用したい場合は、新しいチャットを開始することを推奨します
